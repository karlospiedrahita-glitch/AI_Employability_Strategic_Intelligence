# Inteligencia estratégica para la empleabilidad basada en Inteligencia Artificial

## Descripción

Este proyecto desarrolla un sistema de inteligencia estratégica basado en técnicas de bibliometría, procesamiento del lenguaje natural (NLP), aprendizaje automático (Machine Learning) y modelos de lenguaje de gran escala (LLM) para analizar las tendencias científicas relacionadas con la Inteligencia Artificial y la empleabilidad.

El proyecto integra diferentes metodologías cuantitativas para apoyar la toma de decisiones estratégicas mediante evidencia científica.

---

## Objetivo

Construir un modelo de inteligencia estratégica que permita identificar tendencias, oportunidades y estrategias relacionadas con la empleabilidad basada en Inteligencia Artificial.

---

## Metodología

El proyecto integra seis etapas principales:

1. Preparación y limpieza de datos.
2. Análisis bibliométrico.
3. Análisis de sentimientos mediante BERT.
4. Modelos predictivos de series de tiempo.
5. Dashboard analítico.
6. Generación automática de estrategias mediante modelos LLM.
